> An restaurant menu application to order food using WhatsApp API.

## Technologies

- [TypeScript](https://www.typescriptlang.org/)
- [React Native](https://reactnative.dev/)
- [Tailwind](https://tailwindcss.com/)
- [Expo](https://expo.dev/)

## Install

```sh
npm install
```

## Usage

```sh
npx expo start
```

## Run tests

```sh
npm run test
```
